### Installation

1. Copy `settings/local_settings.examle.py` to `settings/local_settings.py `
2. Set yor own token into `settings/local_settings.py` 
3. Create virtualenv and activate it 
4. `pip install -r requirements.txt` 
5. `python main.py`
